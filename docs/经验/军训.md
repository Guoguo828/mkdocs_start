# 军训经验

## 概述

浙大军训还是很辛苦的，我说实话，军训的分说多不多，说少不少，但是，作为第一个学分开个好头还是很重要的。

但是军训，你只要把基本的东西完成，基本就有不错的分数了，有些东西你想追求也追求不到。

## 军训你必须做到的

1.训练时候认真，至少不要太偷懒
2.不要迟到，早退
3.军训服要穿好，东西配齐
4.训练时候别玩手机，听话
5.宿舍内务做好

## 轻松的加分项
