# 现在我们来学习一下动态规划

## 动态规划

说实话 没什么好讲的 关键要选对状态怎么设计 和状态转移方程

### 步骤

1. **Characterize an optimal solution（刻画最优解的结构）**
   分析问题的最优解具有什么性质，核心是判断是否存在**最优子结构**——即整体最优解包含其子问题的最优解。这是动态规划能应用的前提。

2. **Recursively define the optimal values（递归地定义最优值）**
   将问题的最优值用子问题的最优值**递归表示**，建立“状态转移关系”。这一步会推导出动态规划的核心——**状态转移方程**。

3. **Compute the values in some order（按一定顺序计算最优值）**
   动态规划是**自底向上**计算的，需要确定子问题的计算顺序（比如从小规模子问题开始，逐步推导大规模子问题），避免重复计算，保证效率。

4. **Reconstruct the solving strategy（重构求解策略）**
   若需要得到**具体的最优解（而非仅最优值）**，则需记录每一步的决策，最后通过回溯这些决策，还原出完整的最优解路径。


简单来说，这四个步骤是设计动态规划算法的“标准流程”：先明确最优解的结构，再建立递归关系，接着按序计算，最后（可选）还原具体策略。
